# Playwright Tests for Automation Exercise

Simple automation tests for https://automationexercise.com/ using **Playwright**.

This is my learning project while I'm studying automated testing as a Junior QA Engineer.

## What is this project?

Practice project where I write end-to-end UI tests for a free testing website that has:

- Registration & Login
- Products page
- Cart functionality
- Checkout process
- Contact us form
- ...and many other typical e-commerce features

Goal: learn Playwright basics, page object pattern, good test structure, assertions, waiting strategies, etc.

## Tech Stack

- Playwright
- TypeScript
